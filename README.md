# Forest Fire Prediction using IoT Sensor Data Analytics

## Overview
This project uses machine learning to predict forest fire occurrence from environmental sensor readings such as temperature, humidity, pressure, smoke concentration, CO₂, VOC, and particulate matter.

## Dataset
- Smoke Detection Dataset (Kaggle)
- 62,630 records
- 16 environmental features

## Model
Random Forest Regression

## Performance
- Accuracy: 98.33%
- Precision: 100%
- Recall: 97.67%

## Comparison
Linear Regression Accuracy: 91.67%
Random Forest Accuracy: 98.33%

## Technologies Used

- Scikit-learn
- Pandas

- Matplotlib
